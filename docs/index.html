BitLogic Computer Algebra System

Getting Started

Starting the program

BitLogic.jar is the java executable which can be launched with either the "startJarGui" or "startJarNoGui" scripts. The jar will NOT launch if double clicked. You can also start the program from the terminal using either of the following commands

java -jar BitLogic.jar no-gui
java -jar BitLogic.jar gui

You may need the latest version of the java JRE

Using BitLogic with the terminal

once launched you should get an intro of BitLogic with

>

To quit type "quit" and hit ENTER

Lets solve a simple linear equation: x+3=2x-5 in terms of x
The first thing to know is that 2x must be written as 2*x so that the interpreter knows when to multiply.

Option 1, plain English

> what is solution to x+3=2*x-5 in terms of x

hit ENTER to add it to the stack

**********************
1: solve(x+3=2*x-5,x)
**********************
>

Option 2, BitLogic language

> solve(x+3=2*x-5,x)

hit ENTER to add it to the stack

The English analyzer or BitLogic interpreter created a valid BitLogic expression and added it to whats known as the stack. The stack keeps a list of items which can be manipulated, removed, added, duplicated etc. For the time being just know that this is a view of all the expressions in memory. To get the result of this BitLogic expression type...

result
or
r

hit ENTER
This will compute the last item on the stack and generate the result and replace the old expression. You should get "x=8".
There are many other commands which exist and is under the commands with examples section.
